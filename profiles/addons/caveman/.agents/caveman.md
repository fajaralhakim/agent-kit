# Caveman Integration

[Caveman](https://github.com/JuliusBrussee/caveman) compresses agent context into caveman-speak — fewer tokens per session, same code/URLs/paths preserved byte-for-byte.

**Agent Kit does not run Caveman's installer automatically.** Install Caveman yourself when you want it.

## Install Caveman (opt-in)

Requires Node.js ≥ 18.

```bash
# macOS / Linux / WSL
curl -fsSL https://raw.githubusercontent.com/JuliusBrussee/caveman/main/install.sh | bash

# Windows (PowerShell 5.1+)
irm https://raw.githubusercontent.com/JuliusBrussee/caveman/main/install.ps1 | iex
```

### Narrower options

| Flag | What |
|------|------|
| `--only cursor` | Install for Cursor only |
| `--only cursor --only claude` | Repeat `--only` for multiple agents |
| `--minimal` | Plugin/extension only — no hooks or per-repo rules |
| `--with-mcp-shrink="…"` | Register `caveman-shrink` MCP proxy wrapping an upstream server |
| `--no-mcp-shrink` | Skip MCP shrink registration |
| `--dry-run` | Preview without writing |

Example — Cursor only, minimal:

```bash
curl -fsSL https://raw.githubusercontent.com/JuliusBrussee/caveman/main/install.sh | bash -s -- --only cursor --minimal
```

## Usage in Cursor

- Type `/caveman` or say "talk like caveman" to activate compressed mode.
- Say "normal mode" to stop.
- `/caveman-compress <file>` — rewrite a memory file (e.g. `CLAUDE.md`) into caveman-speak (~46% fewer input tokens).

## Caveman vs Agent Kit memory

| Layer | Purpose |
|-------|---------|
| `.agents/architecture.md`, `.agents/layers/*` | **Project map** — generated by `agent-kit analyze` |
| `.agents/memory/` | **Session/plan memory** — plans, handoffs, decisions across chats |
| Caveman | **Token compression** — shorter phrasing for rules and context files |

They complement each other. Use Agent Kit memory for durable plan state; use Caveman to shrink how much of that state costs per token.
